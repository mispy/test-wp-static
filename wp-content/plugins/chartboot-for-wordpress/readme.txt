== Description ==

ChartBoot for WordPress Plugin allows to create, edit and embed Google Charts in Posts and/or Pages.

For more information, and for premium versions check out [ChartBoot](http://www.chartboot.com/ "ChartBoot")

Steps:

1. Open ChartBoot for Wordpress editor window by clicking on ChartBoot icon media button.
2. "Import Data" pasting from Excel, Calc, etc.
3. If necessary "Set Types" of your Data (i.e., string, number, date). 
4. "Edit Chart" type (e.g., Bar, Scatter, and all other Google Charts types) and set the Look and Feel.
5. Set additional chart styles in "Set Styles". 
6. "Send to WP" and you'll have your Chart shortcode in place.

YouTube tutorial (referring to the 2.4 version, only minor changes in the Set Style section for the 3.0 version):

[youtube http://www.youtube.com/watch?v=ZHz0WvFc3bs]


Enjoy!


== Installation ==

Upload the ChartBoot for wordpress plugin to your blog and Activate it.
